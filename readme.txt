The most responsive and dynamic carousel to showcase your clients, works, and many more.

== Description ==

We created this plugin as a open source standard because we feel like one does not have to hire a developers for making the small things like carousels. Anyone having the ability to manage and edit pages/posts and use shortcodes can easily use this plugin.

SP Clients Carousel provides you with these features:

* Responsive - works well on every devices
* Loops - Loops through the carousel items for infinite time.
* Visit Button - Clickable button which send your user to the respective post related to the carousel items.
* Modern Browsers Support - Owl Carousel does every possible handling by supporting modern browsers.


= Owl Carousel =

[Owl Carousel](https://owlcarousel2.github.io/OwlCarousel2/) is a touch enabled jQuery plugin that lets users create custom carousels.



== Installation ==

1. Upload the entire `sp-carousel-showcase` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu within the WordPress Dashboard.

You will now see the `SP Clients` menu in your WordPress admin panel.

== Frequently Asked Questions ==

= How to insert 'SP Clients Carousel' carousels into my pages/posts? =
* First, Goto the SP Clients section and create a post there. You must add featured image in your post.
* Publish the post.
* You can create as much post as you want in 'SP Clients' section.
* Goto the Page/Post where you want to show the carousel.
* Add [spcarousel_show] shortcode to content where you want to display the carousel.
